# Shootemup

Prototype of a shoot'em up video game.

[![Watch a short gameplay video]](https://youtu.be/luFHxxdOkdY)

## Objective

Survive as long as you can!!! Knights are coming for you

## Install

```markdown
COMING SOON
```

## Control

- `Left click to shoot`
- `ZQSD`to move around
- `Mouse to aim`
- `P` to pause
